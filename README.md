Project: Aqua Sigil
Name: Cù Minh Hải
Description: Turn-based game, di chuyển theo ô. Double click để mở action menu tại chỗ, Right click để mở thông tin nhân vật. Khi unit "Recuit" thăng cấp sẽ chọn 1 trong 3 class "Archer", "Lancer", "Warrior" để promote.
            Tiêu diệt kẻ địch, quả ải mở khóa cốt truyện mới, tăng độ khó. Gồm 6 level, 1 Main menu, 1 Loading Scene, 1 Level Select.
